# 📄 Assignment 1 — CPU Scheduling  

## Algorithms Used  
FCFS (First Come First Serve)  
SJF (Shortest Job First)  

## Concepts  
Completion Time (CT)  
Turnaround Time (TAT = CT − AT)  
Waiting Time (WT = TAT − BT)  

## Output Summary  
FCFS → Simple but higher waiting time  
SJF → Better performance
